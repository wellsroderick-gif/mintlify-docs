The Bringing List gives you a live view of every game your attendees plan to bring to an event, and as an organizer you have full control over it. You can clean up duplicate entries, fix attribution errors, manage table leads, and make corrections even after the event wraps up — ensuring your event record stays accurate.

## Member vs. Organizer Capabilities

Members and hosts share the same Bringing List, but hosts have a broader set of controls:

<CardGroup cols={2}>
  <Card title="Members Can" icon="user">
    - Add their own games to the list
    - Create a table for a game they're bringing
    - Claim seats at any open table
    - Remove their own entries before the event
  </Card>
  <Card title="Hosts Can" icon="shield-check">
    - Everything members can do, plus:
    - Remove any member's game from the list
    - Reassign who is credited as bringing a game
    - Reassign table leads to another attendee
    - Edit bringing list entries for 7 days after the event ends
  </Card>
</CardGroup>

## Removing a Game from the List

To remove a game that a member has added:

1. Open the **Bringing List** tab on your event page.
2. Find the game entry and click the **✕** icon on the right.
3. If the game has an open table with seated members, a confirmation dialog will appear before the entry is deleted.

<Note>
  If you remove a game and its owner was the table lead with other members already seated, those seated members are notified that their seats have been freed and they will need to find another table.
</Note>

## Reassigning Game Attribution

If a game was added under the wrong member's name — or if ownership needs to transfer — you can reassign attribution without removing and re-adding the entry:

1. Click **Change who brought this** on the game entry.
2. Select any attendee with a **Going** RSVP status from the dropdown.
3. Save the change. The selected member is now shown as the one bringing that game.

## Reassigning a Table Lead

Table leads control seating for a game's table. To hand that role to someone else:

1. Open the table attached to the game.
2. Click the table's **action menu** (the ⋯ icon).
3. Select **Change table lead**.
4. Choose a **Going** attendee from the list.

The new table lead immediately takes over management of that table.

## Post-Event Editing

After your event ends, you have a **7-day window** to make corrections to the Bringing List. This lets you fix attribution errors, add games that were brought but never listed, or remove entries that didn't end up coming. After seven days, the list is locked as part of the event's historical record.

<Tip>
  Use the **On The Table** view after your event to see a persistent record of everything that was played at the session — it lives in your event history and contributes to game and member stats over time.
</Tip>
