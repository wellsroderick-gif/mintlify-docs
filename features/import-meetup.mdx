If you're moving your board game group from Meetup.com to WR-Games, you don't have to leave your event history behind. The Meetup import tool maps your existing Meetup events into your WR-Games group so your community can see where you've been — and pick up right where you left off.

## What Gets Imported

WR-Games maps the following information from your Meetup export on a best-effort basis:

| Field | Imported |
|---|---|
| Event title | ✅ Yes |
| Event description | ✅ Yes |
| Date and time | ✅ Yes |
| Location / venue | ✅ Yes (best-effort match) |
| Member list | ❌ No |
| RSVPs | ❌ No |
| Photos | ❌ No |

Members are not automatically imported — they will need to create WR-Games accounts and join your group independently. Past RSVPs and photos are not carried over.

## How to Import Your Meetup Events

<Steps>
  <Step title="Request your Meetup data export">
    On Meetup.com, go to your **group settings** and request a data export. Meetup will prepare a downloadable file containing your group's event history.
  </Step>
  <Step title="Download the export file">
    Once Meetup notifies you that your export is ready, download the file to your device. Keep it somewhere easy to find — you'll upload it in the next step.
  </Step>
  <Step title="Open the import tool in WR-Games">
    In WR-Games, navigate to your **group dashboard → Settings → Import → Import from Meetup**.
  </Step>
  <Step title="Upload your Meetup export file">
    Click **Upload file** and select the Meetup export file you downloaded. WR-Games will parse the file and prepare a list of events for review.
  </Step>
  <Step title="Review and deselect any events you don't want">
    Browse the list of detected events. Deselect any you'd prefer not to import — for example, events unrelated to board games, or duplicates you've already recreated.
  </Step>
  <Step title="Confirm the import">
    Click **Import Events**. WR-Games creates individual event pages for each imported event in your group.
  </Step>
</Steps>

## After the Import

Once the import completes, your events appear in your group as past or upcoming events depending on their original dates. Past events contribute to your group's history, and upcoming events are immediately open for RSVPs.

<Note>
  Meetup import is a one-time migration tool. It does not create an ongoing sync with Meetup.com — any new events you create on Meetup after the import will not automatically appear in WR-Games.
</Note>

<Warning>
  Each import run adds events to your group. If you upload the same export file more than once, you may end up with duplicate event entries. Review your events page after importing before running the tool a second time.
</Warning>

<Tip>
  After importing, share your WR-Games group invite link with your Meetup members so they can join your group on WR-Games and start engaging with your new and imported events right away.
</Tip>
