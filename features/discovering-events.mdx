WR-Games makes it easy to find board game events happening in your area. Whether you are looking for a casual game night, a tournament, or a one-off meetup around a specific title, the platform gives you several ways to surface exactly what you are after — from a broad city-wide browse all the way down to a single game you have been waiting to play with others.

## Browsing Events

Head to the **Events** tab to see a chronological list of all upcoming public events. The page loads with the soonest events first, so you can immediately see what is coming up this week. Each entry in the list is an event card showing you enough detail to decide whether it is worth a closer look.

**What you see on every event card:**

- Date and start time
- Location name and neighbourhood
- Event type (game night, tournament, convention, etc.)
- Current attendee count and total capacity
- Avatars of friends or community members who are already going

## Filtering Events

Use the filter bar at the top of the Events page to narrow the list down to what matters to you. You can combine any of these filters:

| Filter | What it does |
|---|---|
| **Date range** | Show only events starting within a chosen window |
| **Event type** | Game nights, tournaments, conventions, and more |
| **Game title** | Every event that lists a specific game on its table |
| **Location** | A city, neighbourhood, or specific venue |

Filters stack, so you can ask for "all game nights in the East Side this Saturday featuring Wingspan" and get exactly that.

## Search by Game

If you have a game you are itching to play, go straight to the game search on the Events page and type its name. WR-Games returns every upcoming public event that has listed that game — across all event types and venues. This is the fastest way to find a table for a title you own but rarely get to the table.

## Nearby Events

When you set your city in your profile, the Events page can sort results by distance from you. Events closest to your location float to the top, so you are not scrolling past events on the other side of the country. You can still switch back to date order at any time using the sort toggle.

## Discovering Events Through Groups

Some events are private to a group and will not appear in the public Events tab. To find them:

1. Open the **Groups** tab and search for groups in your area or centred on a game you enjoy.
2. Request to join the group.
3. Once accepted, the group's private events appear in your Events feed alongside public ones.

This is the main path to recurring game nights run by local clubs, which are often invite-only.

## Saving Events

When you spot an event you want to keep track of but are not ready to RSVP for, click the **bookmark icon** on the event card or event detail page. Saved events collect under **Events → Saved** so you can come back to them later. The bookmark does not notify the organiser and does not reserve a spot — it is just your personal shortlist.

## Homepage Highlights

The WR-Games homepage surfaces a curated selection of **trending and upcoming events** based on your location and game preferences. It is a good starting point each time you log in, especially if you have not set any filters yet.
